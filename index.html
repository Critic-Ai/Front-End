About Us

This project was made by Ali Qadir, Pratyush Chauhan, Soham Datta. This is a game review website, which scrapes the web for relevant reviews of games.
These reviews are then sent to a sentiment analysis pipeline. For one game, there are ~30 reviews on average, each of these reviews is converted to numerical values between 1-10 based on their predicted sentiment (1 tends to be negative and 10 tends to be positive).
